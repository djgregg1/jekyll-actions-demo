# jekyll-actions-demo

## Description of Project

This project comes from the documentation on using GitHub Actions with Jekyll to create a GitHub Pages website from a GitHub repository.

https://jekyllrb.com/docs/continuous-integration/github-actions/

## Typical Development Workflow

1. From GitHub clone the repository down to a local machine
2. Make any modifications to the local repository through VS Code or equiv.
3. Stage and Commit modifications to local Git repository
4. Push modifications to GitHub repository
5. The push should trigger the GitHub Action & after some time create/update the GitHub Pages website (progress of the GitHub Action can be viewed in the Actions tab of the GitHub repository)
